# ZodErrorType

The type of the data failing validation. It is only retrieved when `code = too_small` or `code = too_big`.


## Values

| Name     | Value    |
| -------- | -------- |
| `String` | string   |
| `Number` | number   |
| `Array`  | array    |
| `Set`    | set      |
| `Date`   | date     |
